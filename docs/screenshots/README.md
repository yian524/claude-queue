# Screenshots

Drop screenshots here and reference them from the top-level `README.md`.

## Recommended shots

| filename | what to capture |
|---|---|
| `queue-ui.png` | the alt-screen queue input with 2-3 pending entries, demonstrating the `Pending (N)` list and `> ` input line |
| `dropdown.png` | the `/` slash-command dropdown while typing `/w` (shows `/wait` highlighted) |
| `direct-mode.png` | normal Claude Code view with status bar showing `claude -q | queue:0 | mode:direct` in the tab title |
| `error-handling.png` | an inline error note (e.g. `unknown command '/5'; try /help`) demonstrating that a bad keystroke no longer kills the session |

## Guidelines

* Use Windows Terminal with a dark theme (default Cascadia Mono, ~14pt).
* Crop to the terminal window only — no taskbar / wallpaper.
* Save as PNG (lossless; better for text rendering than JPG).
* Target width ~1200–1500 px for README embedding.
* Do not include any personal info (user paths are fine; API keys or private session IDs are not).
